(From Software Skills - The Software Developer's Life Manual by John Sonmez)

10-Step Process (Do steps 1 - 6 once; Repeat steps 7 - 10)

  1. Get the big picture

  2. Determine scope

  3. Define success

  4. Find resource

  5. Create a learning plan

  6. Filter resources

  7. Learn enough to get started

  8. Play around

  9. Learn enough to do something useful

  10. Teach

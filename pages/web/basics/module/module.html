Module

  • ES6 define the concept of module
    • The definition favors browser behaviors
  • CommonJS
    • It was a project to establish module spec outside browser, for example, web server or native desktop application
    • Node.JS uses CommonJS's module spec
    • CommonJS can be recognized by require() and module.exports
    • If you need to use "node-style" module (require()) in a browser you need to use transcompiler like browserify and webpack
    • require() is synchronized
  • AMD (Asynchronous module definition)
    • AMD also defines the spec of module, however loading module is asynchronous
    • AMD specification is implemented by Dojo Toolkit, RequireJS, and other libraries
  • UMD (Universal module definition)
    • It attempts to offer compatibility with the most popular script loaders of the day
    • In many cases it uses AMD as a base, with some special case to handle CommonJS compatibility

Difference between module and plain Javascript

  • Module Script Execute in Strict Mode
  • Module Script has its Own Scope (namespace). type=JavaScript has global scope
  • Module Script can Import other Javascript Modules
  • Module Script is Always Deferred

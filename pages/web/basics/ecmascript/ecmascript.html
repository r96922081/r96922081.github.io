ECMAScript (ES)

ECMAScript (or ES) is a general-purpose programming language.   It is a JavaScript standard meant to ensure the interoperability of Web pages across different Web browsers.  ECMAScript is commonly used for client-side scripting on the World Wide Web, and it is increasingly being used for writing server applications and services using Node.js

Package

  •   Example of calling another package

  • Package layout

  • A  name is exported if it begins with a capital letter

  • Fully qualified name of sub-package does not contain it's parent package, for example, you don't call pkg1.pkg2.Hello2 instead you call pk1.Hello2

  • File name does not matter in a package, for example, pkg1.Hello1 is defined in pkg1_whatever.go

  • No module concept in this example.  Module is introduced in later Go version
